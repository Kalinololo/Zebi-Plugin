# Hunger Games Minecraft Plugin

Custom Minecraft server plugin implementing a Hunger Games–style multiplayer game mode.  
Developed as a learning project focused on gameplay systems, multiplayer logic, and event-driven architecture.

## Features
- Lobby management system with player join/leave handling
- Custom command system, including gameplay and fun/utility commands
- Kit selection system allowing players to choose a role at game start
  - Kits grant unique abilities and powers
  - Cooldown management for ability usage
- Player tracking system using a compass
- Death announcements and in-game event handling

## Missing / Incomplete Features
- World management system (automatic world reset or switching between matches)
- Additional kits originally planned but not implemented

## Project Status
Archived learning project.  
The core gameplay systems are mostly functional, but the project is unfinished and no longer actively maintained.
